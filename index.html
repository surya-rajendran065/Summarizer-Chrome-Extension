Briefly Guide

For instructors

*This guide will help you in using this extension*

Activation

How to activate the extension

    Step 1: Press Ctrl + Shift at the same time.
*A side panel will popup*
    Step 2: Hold F2 for 1 sec.
*You will be redirected to a new page*
    Step 3: Click the button on the new page.
*A popup will ask for microphone access*
    Step 4: Click allow microphone use.

Summarization

Summarizes webpage's contents.

Commands

    Ctrl x3 - Start Summarization
    Shift - Change Summary Length
    Caps Lock - Pause/Play Summary

Summary Types

    General - General content + structure of webpage
    Research - Only content of webpage for research
    Paragraph by Paragraph - Summarize each paragraph

Summary Lengths

    Long - 300 words
    Medium - 100 words
    Short - 50 words
    2 sentences - only 2 sentences

A.I Agent

Helps use the browser

Commands

    F2 (1 sec+) - Activate A.I Agent
    Ctrl - Interrupt A.I Agent
    Esc - Kill A.I Agent

Functions

  • Browse Websites: Open websites for you
  • List Tabs: List all your open tabs
  • Answer: Answer general questions like Siri
  • Navigate: Navigate through a website
